JavaScript 有多灵活？

可以在数字原型上扩展方法，还可以通过修改getter和setter来扩展：

Number.prototype = Object.defineProperty(
    Number.prototype, "double", {
        get: function() {
            return (this + this)
        }
    }
);

Number.prototype = Object.defineProperty(
    Number.prototype, "square", {
        get: function() {
            return (this * this)
        }
    }
);

(8).double.square
    // 256
